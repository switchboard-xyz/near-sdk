## Getting Started

To get started, clone the
[sbv2-near](https://github.com/switchboard-xyz/sbv2-near) repository.

```bash
git clone https://github.com/switchboard-xyz/sbv2-near
```

## SDKs

- [Rust](/rust/sbv2-near/): Rust crate to deserialize and read Switchboard data
  feeds
- [Typescript](/javascript/near.js/): Typescript package to interact with
  Switchboard V2

## Programs

- [feed-parser](/programs/feed-parser/): Read a Switchboard feed on NEAR
